# Iris Classification using KNN and SVM

This project demonstrates machine learning classification on the Iris dataset using two algorithms: K-Nearest Neighbors (KNN) and Support Vector Machine (SVM).

# Features

Data preprocessing and feature scaling

Train / Validation / Test split (60% / 20% / 20%)

Hyperparameter tuning using Validation set

Final evaluation using Accuracy, Precision, and Recall

Visualization of decision boundaries (SVM)

# Files

knn_main.py – Iris classification using KNN with hyperparameter tuning for optimal K.

svm_main.py – Iris classification using SVM with kernel and C tuning, including PCA visualization.

# Dependencies

numpy

scikit-learn

matplotlib

pandas

# Dataset

The project uses the Iris dataset provided by scikit-learn.

# Run
python knn_main.py
python svm_main.py
